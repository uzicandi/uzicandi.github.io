```cs
- 배열
- 리스트
- 스택 : 컴파일러, 운영체제 구현에서 폭넓게 사용
- 큐 : 은행이나 슈퍼에 줄 선 사람들을 추상화 한 구조. 차례로 데이터 처리
- 연결리스트 : 리스트 자료구조를 변형시킨 형태, 각 객체끼리 연결되어 있는 자료구조
- 연결 리스트는 데이터를 자주 [삽입하거나 삭제]하는 프로그램에 효율적인 자료구조
- 딕셔너리: 키와 값 쌍으로 데이터를 저장
  - 해시테이블, 해시 알고리즘을 이용해 테이블에 데이터를 저장
- 세트 자료구조: 중복된 데이터를 포함하지 않은 상황에 적합
- 이진 트리와 이진 탐색 트리 : 정렬된 형태로 데이터를 저장해야 할때
- 그래프, 그래프 알고리즘 : 노드, 지도의 도시 등과 같은 데이터를 표현하는데 사용
- 자료구조가 아닌 알고리즘: 작은 데이터를 처리하는데 적합, 구현하기 쉬운 간단한 정렬 알고리즘, 대용량 데이터 세트 처리
- 순차 검색, 이진 검색: 검색 관련 알고리즘
- 동적 프로그래밍, 탐욕 알고리즘 같은 고급 알고리즘: 기존 알고리즘으로는 너무 느리거나 구현하기가 어려울때

```
