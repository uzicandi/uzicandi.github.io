### Next/Image

- 이미지 최적화 컴포넌트
- 이미지 처리를 간소화하고 성능을 향상시키는데 도움을 줌

1. 자동 이미지 최적화

- 다양한 디바이스와 화면 크기에 맞춰 이미지를 자동으로 최적화

2. 지연로딩

- 뷰포트에 들어올 때만 이미지를 로드하여 초기 페이지 로드 시간을 줄임

3. 다양한 포맷 지원

- WebP 같은 최신 이미지 포맷을 자동으로 제공

4. 레이아웃 시프트 방지

- 이미지 로딩 중 레이아웃 시프트를 방지하여 사용자 경험을 개선

5. CDN 지원

- 외부 이미지 소스 또한 자동 최적화를 제공

### Next/Link

- next.js에서 제공하는 CSR을 위한 컴포넌트
- SPA같은 사용자 경험을 제공하면서도 SSR의 이점을 유지할 수 있음

1. CSR: 페이지 전체를 새로 로드하지 않고 필요한 컴포넌트만 업데이트
2. 자동 Prefetch: 뷰포트에 Link 컴포넌트가 들어오면 해당 페이지의 JS 파일을 자동으로 미리 로드
3. 코드 스플리팅: 각 페이지 별로 필요한 JavaScript만 로드하여 초기 로딩시간을 줄임
4. 동적 라우팅 지원: 동적 세그먼트를 포함한 라우트를 쉽게 처리할 수 있음
5. 외부 링크 지원: 외부 URL에 대해서는 일반 `<a>` 태그처럼 작동

장점

- 빠른 페이지 전환
- 더 나은 SEO 성능
- 앱의 전반적인 성능 향상
